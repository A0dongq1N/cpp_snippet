# Indicators Library Demo

This is a simple demo of the [indicators](https://github.com/p-ranav/indicators) library, which provides activity indicators like progress bars and spinners for Modern C++.

## Prerequisites

- CMake 3.14+
- C++11 compliant compiler
- Internet connection (to fetch the library via FetchContent)

## Build and Run

```bash
mkdir build
cd build
cmake ..
make
./indicators_demo
```
